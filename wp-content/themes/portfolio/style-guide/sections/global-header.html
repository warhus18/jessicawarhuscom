Global Header

SCSS Location: /styles/scss/partial/_global-header.scss

Description

A universal header with supporting utility links, messages, and account details that apply to all areas of the application.

Guidelines

  • An application’s name should follow the Vertex logo.
  • Keep navigation links short. Users should be able to identify links without relying on color. Underline is removed on hover.
  • Second level Account links may be housed under a drop-down menu.

Usage

  • Single-use component with text for each specific brand.
  • An optional utility navigation can be displayed with a small select group of links.
